# PowerShell AD Tools

Coleção de scripts PowerShell para administração e automação do Active Directory.

## Recursos

- Criação de usuários
- Reset de senha
- Desbloqueio de contas
- Gestão de grupos
- Relatórios administrativos
- Onboarding e Offboarding

## Estrutura

/users → scripts de usuários  
/groups → gestão de grupos  
/reports → relatórios administrativos  
/onboarding → automação de admissão  
/offboarding → desligamento de usuários

## Tecnologias

- PowerShell
- Active Directory Module
- Windows Server

## Objetivo

Automatizar tarefas recorrentes de administração de ambiente corporativo.
